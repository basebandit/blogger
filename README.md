# Blogger

> Blogger is a simple golang http server that serves html files from a directory.


## How to use it
- Download the project from github  

```go
go get -u github.com/basebandit/blogger
```  

- Run it  

```bash
└──╼ $ blogger -port 8888 
Server listening on address :8888
```
